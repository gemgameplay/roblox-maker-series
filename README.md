# roblox-maker-series
GemGame Roblox Dev Kit — 가이드·코드·PPT 자료 모음

`gemgame` 유튜브(젬겜) 로블록스 만들기 시리즈의 공개 자료 저장소입니다.  
각 게임(EP)마다 **제작 가이드**, **소스코드**, **로드맵 PPT**, **코드 설명 PPT**를 한 폴더에 묶어 제공합니다.

---

## 빠른 다운로드 방법

### 1) 전체 자료 한 번에 받기
- 이 저장소 페이지에서 **Code → Download ZIP** 으로 전체를 내려받을 수 있습니다.

### 2) 특정 EP만 받기(권장)
- EP별로 내려받고 싶다면, 각 EP 폴더의 안내(README) 또는 Release(추후 제공)에서 ZIP을 받으세요.
  - (운영 예정) **Releases**에 EP별 ZIP을 올려 “EP 단위”로 바로 다운로드 가능하게 하겠습니다.

---

## 폴더 구조

각 EP(게임)는 아래 구조로 정리합니다.

/games/EP01_<GameName>/
01_guide/ # GPT가 알려준 게임 만드는 법(설명/문서)
02_source/ # 소스코드
03_roadmap_ppt/ # 게임만들기 로드맵 PPT
04_codewalk_ppt/ # 소스코드 설명 PPT
README.md # 해당 EP 안내(실행 방법/학습 포인트/파일 설명)


---

## 에피소드 목록 (업데이트 중)

| EP | 게임명 | 핵심 학습 포인트 | 폴더 |
|---:|---|---|---|
| EP01 | (예: Obstacle Dodge) | (예: 장애물 회피, 충돌 처리, UI) | `games/EP01_ObstacleDodge/` |
| EP02 | - | - | - |

※ 각 EP 폴더의 `README.md`에 **설치/실행 방법**, **학습 목표**, **파일별 역할**을 상세히 적어두겠습니다.

---

## 포함 자료 안내

- **가이드(01_guide)**  
  게임 제작 절차, 핵심 개념, 구현 순서, 체크리스트
- **소스코드(02_source)**  
  로블록스 스튜디오에서 재현 가능한 예제 코드(스크립트/모듈/UI 등)
- **로드맵 PPT(03_roadmap_ppt)**  
  “이 게임에서 무엇을 배우는지 / 다음 단계로 어떻게 확장하는지” 학습 흐름
- **코드 설명 PPT(04_codewalk_ppt)**  
  주요 코드의 의도, 구조, 초심자 관점 해설(왜 이렇게 작성했는지)

---

## 권장 사용 방식(초심자 기준)

1. 해당 EP 폴더로 이동 → `README.md` 먼저 읽기  
2. `01_guide`로 전체 흐름 이해  
3. 로블록스 스튜디오에서 기본 맵/오브젝트 구성  
4. `02_source`의 코드를 순서대로 적용  
5. `04_codewalk_ppt`로 코드 구조 이해 → 내 게임으로 확장
